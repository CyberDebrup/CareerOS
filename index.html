Deploy Guide

Go Live in 6 Steps

1
Get a Domain
Go to Namecheap or Google Domains. A .dev or .tech domain costs ~₹800–₹1500/year and looks sharp for dev projects.
2
Push to GitHub
Create a repo → git init → git add . → git commit -m "init" → git push origin main
3
Deploy Frontend on Vercel
Go to vercel.com → Import GitHub repo → Click Deploy. ~60 seconds. Add custom domain in Settings → Domains.
4
Deploy Backend on Railway
Go to railway.app → New Project → Deploy from GitHub. Add: ANTHROPIC_API_KEY, MONGODB_URI, JWT_SECRET.
5
Get Anthropic API Key
Sign up at console.anthropic.com → API Keys → Create. Add to Railway env only — never in frontend code.
6
Connect & Go Live 🎉
Add CNAME record in domain registrar: www → cname.vercel-dns.com. Vercel auto-configures SSL. You're live!
